# 🤖 AI Data Analyst Agent

An intelligent **Agentic AI project** that automatically analyzes datasets, generates visualizations, detects trends, and produces AI-generated reports in **PDF and PowerPoint format**.

This project demonstrates how **AI agents can automate the role of a data analyst** by performing data cleaning, visualization, insight generation, and report creation.

---

# 🚀 Features

• Upload any CSV dataset

• Automatic **data cleaning**

• **Auto chart selection** based on dataset

• Data **trend detection**

• **Highest & lowest value detection**

• AI-generated insights

• **PDF report generation**

• **PowerPoint presentation generation**

---

# 🧠 Agent Workflow

The AI Data Analyst Agent performs the following tasks automatically:

1️⃣ Clean the dataset

2️⃣ Select appropriate chart type

3️⃣ Generate visualizations

4️⃣ Detect patterns and trends

5️⃣ Generate AI insights

6️⃣ Create downloadable reports

# ⚙️ Installation

Clone the repository

```
git clone https://github.com/YOUR_USERNAME/ai-data-analyst-agent.git
```

Go to project folder

```
cd ai-data-analyst-agent
```

Install dependencies

```
pip install -r requirements.txt
```

Run the application

```
streamlit run app.py
```

---

# ☁️ Deployment on AWS EC2

1. Launch an EC2 instance

2. Connect using SSH

```
ssh -i aws-key.pem ubuntu@your-public-ip
```

3. Clone repository

```
git clone https://github.com/YOUR_USERNAME/ai-data-analyst-agent.git
```

4. Install dependencies

```
pip install -r requirements.txt
```

5. Run Streamlit server

```
streamlit run app.py --server.port 8501 --server.address 0.0.0.0
```

Access the application

```
http://YOUR_PUBLIC_IP:8501
```
